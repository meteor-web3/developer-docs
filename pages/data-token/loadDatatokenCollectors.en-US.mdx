# loadDatatokenCollectors

Get the collector addresses of a data token.

## Usage

```js
await connector.runOS({
  method: SYSTEM_CALL.loadDatatokenCollectors,
  params: datatokenId,
});
```

## Parameters

- `datatokenId`: `string` - data token id.

## Returns

Collector list.

Example:

```json
[
  {
    "collector": "0x312eA852726E3A9f633A0377c0ea882086d66666",
    "dataToken": {
      ...
    },
    "collect_nft": "0xBE87fB2Aa6018d243ac36c745d844A9FEe656D2b",
    "collect_nft_token_id": "1",
    "created_at": "2023-11-01 21:09:26.503328 +0800 CST"
  },
  ...
]
```
