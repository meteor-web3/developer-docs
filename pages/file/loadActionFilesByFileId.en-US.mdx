# loadActionFilesByFileId

Give an indexFileId, load the actionFiles linked to it.

## Usage

```javascript
await connector.runOS({
  method: SYSTEM_CALL.loadActionFilesByFileId,
  params: indexFileId,
});
```

## Parameters

- `indexFileId`: `string` - id of an indexFile

## Returns

`Promise<Record<string, MirrorFile>>` - a map of actionFileId to actionFile
