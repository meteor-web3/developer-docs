# unlockFile

Unlock the content of a monetized file you bought.

## Usage

```js
await connector.runOS({
  method: SYSTEM_CALL.unlockFile,
  params: indexFileId,
});
```

## Parameters

- `indexFileId`: `string` - The index file id if you are unlocking a file

## Returns

```js
{
  fileContent: {
    file: Omit<MirrorFile, "fileKey" | "content" | "external">;
    content: FileContent | string;
  };
}
```

<Callout>
  This function returns exactly the same as{' '}
  <a href="/system-calls/collectFile">collectFile</a> but with the file content
  unlocked.
</Callout>
